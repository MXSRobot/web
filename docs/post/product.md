# 我们的产品

## 可拼组多形态智能机器人
1.积木拆解、拼组成车、人、动物、飞行器等各种机器人；

2.地面机器人具有运动，转动，抓取，检测，避障等功能;

3.飞行机器人具有多种飞行模式（室内定点飞行、避障飞行、追踪飞行）;

4.所有形态机器人均可图形化编程控制，任意添加传感器和执行器，扩展出各种意想不到的功能。
